Personal portfolio website built with **React** and **Vite**, deployed to **GitHub Pages**.

## Development

```bash
# Install dependencies
bun install

# Start dev server
bun run dev

# Build for production
bun run build

# Preview production build
bun run preview
```

## Linting & Formatting

```bash
# Lint
bun run lint
bun run lint:fix

# Format
bun run format
bun run format:check
```

## Deployment

Deployed automatically via GitHub Actions on push to `main`. See [.github/workflows/deploy.yml](../.github/workflows/deploy.yml).
