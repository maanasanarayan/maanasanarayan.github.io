Academic Projects

Sep 2021 - Nov 2021

Bug Bash - An Automation Tool
Amazon, Seattle
  • Bug Bash is a process followed in the Amazon's Choice team to test the AC Badging functionality before deploying a new change. This process is done manually taking up about 8+ hours of engineering time.
  • This project aimed at automating this process by providing an interface for the AC engineers to interact with, and scraping the amazon.com website automatically to get the results.
  • The project started off with Requirements gathering phase, where I disucssed with multiple stakeholders and worked on a requirements document. Following this was the design phase where I jotted down all the design choices.
  • The chosen design consisted of a front-end written in ReactJS and hosted on Amazon's Harmony console. The back-end was written in NodeJS as an AWS Lambda function. The backend system also used an API Gateway to communicate with React app and AWS S3 to fetch the input file.
  • The web scraping logic was written using the NodeJS library, Puppeteer for headless browsing using Chrome.
  • Once the logic was developed, it was thoroughly tested using Jest Unit Tests and manual end-to-end tests.
  • The complete project was built and tested in the 12-week rigorous timeline
